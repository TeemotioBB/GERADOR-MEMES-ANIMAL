# Música fixa no Cretino Factory

A faixa **La Isla Bonita — Madonna** está salva em:

`music/la-isla-bonita.mp3`

Na tela, em **Música do vídeo**, escolha:

- **La Isla Bonita — Madonna** para gerar o MP4 com áudio;
- **Sem música** para gerar o MP4 mudo.

O volume padrão é 20% e pode ser alterado na própria tela. A opção vale tanto para o botão verde de download individual quanto para a geração do lote completo.

## Adicionar outra música depois

1. Coloque o novo MP3 dentro da pasta `music/`.
2. Abra `config.py`.
3. Adicione uma entrada em `MUSIC_TRACKS`, seguindo o exemplo já existente.

A nova música aparecerá automaticamente no seletor da tela.
